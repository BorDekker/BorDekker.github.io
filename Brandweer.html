Fire brigade project

What is this project about?

In this project I made 3D assets for the fire brigade. I made the back of an fire brigade car. With this device they can use it to control the multiple functions of the car such as the sirens. They can also check how much fuel or water there is left.

Project date: November 2024 - January 2025

Project duration: 6 weeks

Development Team: Solo project

Engine: None

Software: Blender

Programming language: None

My role & contribution in this project

Creating 3D assets for the fire brigade.
