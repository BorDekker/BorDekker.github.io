Marbles & Cup project

What is this project about?

In this project I wanted to use the basics of Blender to make an simple modular dungeon. I can reuse all assets to shape my dungeon into whatever size or shape I want. It was a fun and enjoyable project to do.

Project date: November 2024

Project duration: 2 weeks

Development Team: Solo project

Engine: None

Software: Blender

Programming language: None

My role & contribution in this project

Creating a modular dungeon.
